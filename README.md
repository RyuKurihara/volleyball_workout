# volleyball_workout

バレーボールをもっと楽しくプレーしたい僕が設定した、バレーを楽しむためのトレーニングPJ4つ。この4つを高い水準で継続できればある程度高いレベルでバレーボールを楽しめる想定。

> 僕にとって楽しいバレー = 「自分のやりたいプレー動作を実現してスゲェと言われること」なおかつ「自分のプレーが勝因となって試合に勝つこと」
