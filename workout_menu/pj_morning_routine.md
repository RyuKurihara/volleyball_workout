# 朝ルーチン - 週6回実施

> 環境: 自宅マンション（静音必須）or 屋外（硬め公園・アスファルト）  
> 日によって内容が違う理由: 夜トレの種類に合わせた準備。高衝撃ジャンプは公園の硬い床ではNG。  
> カーフ（腓腹筋・ヒラメ筋）は回復が速いため毎朝実施してOK。

---

## 朝A: 足部・足首・股関節（月・木）

> 夜が脚トレなので、足首・股関節・前腿を先に解放しておく日。所要時間: 約30分

### 足部ケア

| 種目 | レップ/時間 | 備考 |
|------|-----------|------|
| 足趾ほぐし | 2分 | 母趾・小趾・2〜4趾それぞれ動かす |
| トー・ヨガ | 10回×2セット | 母趾だけ上げる / 他4趾だけ上げる |
| ショートフット | 5秒×10回 | 指を丸めない。親指球・小指球・踵の3点で土踏まずを短くする |
| 片脚ショートフット保持 | 5秒×5回/脚 | 安定したらカーフレイズ中にも意識 |

### 足首・下腿

| 種目 | レップ/時間 | 備考 |
|------|-----------|------|
| クーチストレッチ（足首背屈） | 45秒×2/脚 | 踵を浮かせない |
| スタンディングカーフレイズ（両脚） | 15回×2セット | 下で1秒止め、上で絞る |
| 片脚カーフレイズ | 10回×2/脚 | 腓腹筋。慣れたら壁なしで |
| シーテッドカーフレイズ（椅子で代用可） | 15回×2セット | 膝を曲げてヒラメ筋メイン |

### 股関節・臀部

| 種目 | レップ/時間 | 備考 |
|------|-----------|------|
| 90/90ストレッチ | 45秒×2/脚 | |
| ヒップサークル | 10回×2/脚 | |
| グルートブリッジ（片脚） | 10回×2/脚 | 臀部を意識して押し上げる |
| クーチストレッチ追加 | 45秒/脚 | 前腿〜股関節前面の伸張 |

### 膝の安定性メンテナンス（器具なし）

> 清水選手のような膝のケガを防ぐための予防ルーティン。特に着地時の膝外反（内側への崩れ）を防ぐ。

| 種目 | レップ/時間 | 備考 |
|------|-----------|------|
| 片脚バランス保持 | 30秒×2/脚 | まず両目で。慣れたら目を閉じる |
| サイドライイングヒップアブダクション | 15回×2/脚 | 中殿筋。膝外反防止の核心種目 |
| クラムシェル（仰向け、膝曲げ・股関節外旋） | 15回×2/脚 | 股関節外旋筋群 |
| スロースクワット（3カウントダウン） | 8回×2セット | 膝がつま先と同方向に追従しているか確認 |
| ウォールシット | 30秒×2セット | 大腿四頭筋の等尺性強化 |

## 入浴中メンテナンス（月・水・木・金確実、土日ボーナス）

> 所要時間: 7分。火曜はカウントしない。周りの迷惑にならない静かな動きのみ。

| 時間 | 内容 |
|------|------|
| 0:00〜1:00 | 足趾を揉む（母趾・小趾・2〜4趾をそれぞれ） |
| 1:00〜2:00 | 足裏の土踏まず起始・終点を押しながら足趾曲げ伸ばし |
| 2:00〜3:00 | 足趾をMP関節から底屈・背屈 |
| 3:00〜4:00 | ショートフット感覚練習（指を丸めず土踏まずを軽く引き上げ） |
| 4:00〜5:00 | 前腿を摘まんで揺らす（強く潰さない） |
| 5:00〜6:00 | 腹直筋横あたりを摘まんで呼吸（吐く時に緩める） |
| 6:00〜7:00 | 目線移動（顔固定で上下左右・斜めに目だけ動かす） |
